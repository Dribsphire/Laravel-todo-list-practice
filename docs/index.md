# Laravel Todo List Practice

A small Laravel app that lets you create and manage a list of tasks.

## What’s included

- Add a task (title)
- List tasks
- Delete a task
- (Code includes edit/update routes)

## Quick start (local)

```bash
composer install
copy .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```

Then open `http://127.0.0.1:8000`.
